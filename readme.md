# Sample API Requests

## Return all restaurants in database:
http://localhost:5000/api/v1/restaurants

## Return restaurants in a certain ZIP code - page 1 of results:
http://localhost:5000/api/v1/restaurants?zipcode=10012

## Return restaurants in a certain ZIP code - page 2 of results:
http://localhost:5000/api/v1/restaurants?zipcode=10012&page=2

## Return restaurants with Italian cuisine:
http://localhost:5000/api/v1/restaurants?cuisine=Italian

## Return restaurants with Riviera in the name:
http://localhost:5000/api/v1/restaurants?name=Riviera
